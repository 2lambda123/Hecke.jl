    Abelian Groups

    Here we describe the interface to abelian groups in Hecke.

    Introduction

    Within Hecke, abelian groups are of generic abstract type GrpAb which does not have to be finitely generated, is an example of a more general abelian group. Having said that, most of the functionality is restricted to abelian groups that are finitely presented as -modules.

    Basic Creation

    Finitely presented (as -modules) abelian groups are of type GrpAbFinGen with elements of type GrpAbFinGenElem. The creation is mostly via a relation matrix for and . This creates a group with generators and relations

    Direct Products

    direct_product(G::GrpAbFinGen...; task::Symbol = :sum) -> GrpAbFinGen, GrpAbFinGenMap, GrpAbFinGenMap

    Returns the direct product of the abelian groups . task can be ":sum", ":prod", ":both" or ":none" and determines which canonical maps are computed as well: ":sum" for the injections, ":prod" for the projections.
